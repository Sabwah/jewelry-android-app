Jewelry Store App

A modern Android app for browsing and managing jewelry products, built with Kotlin and Firebase. Designed for both customers and admins to provide seamless shopping and inventory management.

What It Does

Allows users to browse jewelry items by category (Necklaces, Bracelets, Earrings, Rings, Bags, Watches)

Customers can add items to their cart and proceed to checkout

Admins can log in, upload new items, edit existing products, and manage inventory

Real-time updates using Firebase Firestore

Features

User Authentication: Secure login and registration with Firebase

Product Catalog: Browse items by categories with images and stock status

Cart & Checkout: Add items to cart, view summary, and complete purchase

Admin Panel: Upload, edit, and delete products (image uploads via Cloudinary)

MVVM Architecture: Clean separation of UI and logic for maintainable code

Responsive Design: Smooth, user-friendly UI with Jetpack Compose

Tech Stack

Language: Kotlin

Architecture: MVVM

Backend: Firebase Authentication & Firestore

Storage: Cloudinary for images

UI: Jetpack Compose
